# Project Atom — Embedded Agentic Payroll

A product concept site demonstrating how autonomous AI agents can be embedded inside banking apps to transform the payroll experience for UK SMEs.

## Tech Stack

- **React 18** + TypeScript
- **Vite** — build tooling
- **Tailwind CSS** — utility-first styling
- **Framer Motion** — animations
- **React Router** — client-side routing

## Getting Started

```sh
npm install
npm run dev
```

## Pages

- `/` — Landing page with product narrative, comparison table, UX flow, and pitch
- `/demo` — Interactive 6-step walkthrough with phone mockup

## Scripts

- `npm run dev` — Start dev server
- `npm run build` — Production build
- `npm run lint` — Run ESLint
- `npm run test` — Run tests
